# Why Docuflow?

That's a good question! Here are some reasons why you might want to use Docuflow:

- **Easy to Use**: Seriously, you can get working documentation up and running within
1 minute!

- **Fast Development**: With Hot Module reloading, you can see the changes you make in real time. 

- **Customizable**: Docuflow is highly customizable, allowing you to easily change the look and feel of your documentation site to match your project's branding.

- **SEO Friendly**: Docuflow generates static HTML files, which makes it SEO friendly and easy for search engines to crawl and index your documentation site.

- **Open Source**: Docuflow is open source and free to use, so you can contribute to the project and help make it better for everyone.

- **Documentation**: Docuflow has extensive documentation to help you get started and make the most of the tool, so you can quickly get up to speed and start writing your project's documentation.

- **Continuous Updates**: Docuflow is actively maintained and updated, so you can be sure that you're always using the latest and greatest version of the tool.

- **Responsive Design**: Docuflow generates a responsive documentation site that looks great on all devices, so your users can access your documentation from anywhere.

- **Dark Mode**: Docuflow supports dark mode out of the box, so your users can switch between light and dark themes to suit their preferences.
